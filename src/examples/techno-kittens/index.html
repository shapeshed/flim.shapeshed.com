Kittens love HTML 5

This is a demo of the HTML5 Canvas element

The design is pants but the technology is good because:

  • There are no third-party plugins like Flash or Silverlight needed.
  • It is made using open standards
  • You can view the source
  • It uses open standards so machines and people will be able to easily read and understand it.

If you are interested in learning more about HTML5 I recommend that you watch Ian Hickson's talk. The code for this example was largely based on the example Ian gives in the video.
